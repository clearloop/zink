# The zink compiler

## Installation

```
cargo install zinkup --features zinkc
```

## Usage

```bash
The Zink Compiler

Usage: zinkc [OPTIONS] <INPUT>

Arguments:
  <INPUT>  The path of the wasm file

Options:
  -o, --output <OUTPUT>  Write output to <filename>
  -v, --verbose...       Verbose mode (-v, -vv, -vvv, etc.)
  -h, --help             Print help
  -V, --version          Print version
```
